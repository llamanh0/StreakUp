# 🔥 StreakUp - Motivation Tracking App

> **Boost your motivation, reach your goals, succeed together with your friends!**

StreakUp is a mobile application that helps users increase their motivation by tracking their study hours and streaks, encouraging collaborative work through group support.

## 📱 Application Overview

### 🎯 Purpose
- Track users' daily study habits
- Maintain motivation through streak system
- Provide social motivation through group support
- Support personal development goals

### 👥 Target Users
- University students
- Exam preparers
- Personal goal trackers
- Anyone who wants to work with group support

## ✨ Key Features

### 🔐 User Management
- Secure login with Firebase Authentication
- Email/password and Google sign-in options
- Personal profile management

### 📊 Study Tracking
- Daily study time recording
- Automatic streak calculation
- Personal statistics and charts
- Weekly/monthly progress tracking

### 👥 Group System
- Create groups with friends
- Join groups with invite codes
- View group members' streaks
- Group motivation feed

### 🏆 Motivation System
- Streak milestones
- Achievement badges
- Daily goal tracking
- Motivational messages

## 🛠️ Technology Stack

### Frontend
- **Flutter** - Cross-platform mobile development
- **Dart** - Programming language
- **Material Design** - UI components
- **Provider** - State management
- **Go Router** - Navigation management

### Backend
- **Firebase Authentication** - User management
- **Firestore Database** - Real-time NoSQL database
- **Firebase Cloud Functions** - Server-side logic
- **Firebase Cloud Messaging** - Push notifications

### Design
- **Figma** - UI/UX design and prototyping
- **Material Design 3** - Design system guidelines

## 🚀 Quick Start

### 1. Design Phase
```bash
1. Create Figma account
2. Follow FIGMA_WIREFRAME_GUIDE_EN.md
3. Create design system
4. Draw wireframes for main screens
```

### 2. Development Environment
```bash
1. Install Flutter SDK
2. Install Android Studio
3. Install VS Code with Flutter extensions
4. Create Firebase account
```

### 3. Project Setup
```bash
1. flutter create streak_up
2. cd streak_up
3. flutter pub add firebase_core
4. flutter pub add firebase_auth
5. flutter pub add cloud_firestore
6. flutter pub add provider
```

## ❓ FAQ (Frequently Asked Questions)

### Q: Can I complete this project with zero knowledge?
**A:** Yes! With the planned 12-week roadmap, you can learn step by step and complete it.

### Q: Which platform should I develop for?
**A:** With Flutter, you can write one code for both Android and iOS. Focus on Android for the beginning.

### Q: Do I have to use Firebase?
**A:** Firebase is the easiest backend solution. Alternatively, you can use local storage, but group features become difficult.

### Q: I have no design experience, what should I do?
**A:** Follow the Figma guide and use Material Design 3 guidelines. Start with ready-made components.
